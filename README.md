## 🚀 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/kamran-dev/WebUI-Crafters.git
   ```

2. Navigate into the folder:
   ```bash
   cd WebUI-Crafters
   ```

3. Open any day’s folder in your browser using VS Code Live Server or directly in the browser.
